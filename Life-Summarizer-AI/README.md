# Life Summarizer AI

Life Summarizer AI is a full-stack AI application that helps users reflect on their daily life by writing journal entries and receiving AI-generated summaries, sentiment analysis, and weekly emotional trends.

---

## Features
- Write daily journal entries
- AI-generated text summarization
- Sentiment analysis (positive, neutral, negative)
- Weekly mood trend visualization
- Secure encrypted storage of journal data
- Web interface using React and Flask

---

## Tech Stack
Frontend:
- React.js
- Recharts

Backend:
- Python 3.10
- Flask
- Pandas

AI / ML:
- NLP preprocessing
- Text summarization
- Sentiment analysis

Security:
- Fernet encryption

## How to Run the Project

### Backend
cd backend
python app.py
Backend runs on: http://localhost:5000

### Frontend
cd frontend
npm start
Frontend runs on: http://localhost:3000
