## 🎨 BRAND VOICE GUIDELINES

### Tone: Conversational + Authoritative
✓ Use contractions ("you'll", "it's", "we're")  
✓ Address users directly ("you", "your")  
✓ Avoid jargon—make AI/sentiment analysis feel approachable  
✓ Balance warmth with confidence  

### Style: Clear + Poetic
✓ Short sentences when revealing benefits  
✓ Metaphors that resonate (weather, journeys, maps, clarity)  
✓ Active voice  
✓ Avoid corporate-speak  

### Examples
**❌ Don't Say:**
```
Our application utilizes advanced NLP algorithms to process user-generated content.
```

**✓ Do Say:**
```
Our AI reads your words to find what matters most.
```

---

**❌ Don't Say:**
```
Key differentiators include encryption protocols.
```

**✓ Do Say:**
```
Your thoughts stay private—that's not a feature, it's a promise.
```

## 📝 NOTES FOR WRITERS

- Emphasize **emotional benefits** over technical features
- Use **first-person language** when possible ("your mood", "your journey")
- Create **narrative momentum**—each message builds on the last
- Test copy with your **target user** before publishing
- Keep sentences **short and punchy** in UI
- Reserve longer, poetic copy for **marketing pages**
- Always explain the **"why"** behind features, not just the "what"
